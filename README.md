# mcp-example
Springboot app with example MCP tools


# MVN Command to build
mvn clean package -DskipTests


# Java Command to run the server
java -jar /mcp-server-spring-java/target/mcp-spring-java-0.0.1-SNAPSHOT.jar

## Alternatively, you may use
mvn spring-boot:run


# MCP Configuration
Once the deployment is successful, add MCP server using URL http://localhost:8090 with your host application.
